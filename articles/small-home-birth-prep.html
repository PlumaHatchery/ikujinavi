PREPARATION GUIDE

狭い家・収納少なめの出産準備リスト

最初から全部そろえず、退院直後に使うもの・サイズを測ってから決めるもの・あとで足せるものに分けて考えます。

公的情報確認日: 2026-05-12 / 参考: 厚生労働省 出産なび、消費者庁 こどもの事故防止

結論

狭い家の出産準備は、少なめに始めて生活に合わせて足す。

置き場所が限られる家では、「あると便利」より「退院直後に本当に使うか」を先に見ます。

すぐ使う サイズを測る あとで足す 収納を増やしすぎない

SORT FIRST

まず3つに分ける

01退院直後から使う

おむつ、肌着、授乳まわり、寝る場所など。

02サイズを測って決める

ベッド、ベビーカー、ベビーバスなど大きいもの。

03生活後に足す

おむつ用ゴミ箱、ベビーモニター、大型マットなど。

STARTER SET

先に用意したいもの

衛生

おむつ・おしりふき

最初は少量から。サイズアウトや肌に合うかを見て買い足します。

衣類

肌着・服・おくるみ

洗濯頻度で必要枚数が変わるため、収納に収まる量から始めます。

授乳

授乳・ミルク関連

授乳方法は出産後に変わることがあります。最低限＋買い足しやすさで考えます。

睡眠

赤ちゃんの寝る場所

通路、エアコン、窓、家具との位置関係を確認。安全条件は公式情報を見ます。

沐浴

最低限の沐浴用品

使う場所、乾かす場所、しまう場所を先に決めます。

退院

移動に必要なもの

車、タクシー、徒歩、公共交通機関で必要な用品が変わります。

MEASURE BEFORE BUYING

サイズを測ってから決めるもの

睡眠大型

ベビーベッド

外寸だけでなく、扉や引き出しを開けた時のスペース、周囲を通れる幅を確認。使用時は柵・収納扉のロックなど、メーカー説明と公的な注意喚起も見ます。

見る点
ミニサイズ、折りたたみ式、レンタルも候補。安全マーク・対象月齢・使用条件は購入前に再確認
外出玄関

ベビーカー

玄関や廊下に置くなら、折りたたみ後の幅・奥行き・高さを先に確認。

見る点
自立、持ち運び、電車移動での扱いやすさ
沐浴短期

ベビーバス

使用時サイズだけでなく、乾かす場所と収納場所を確認。

見る点
折りたたみ式、空気式、シンク利用タイプ
居場所床面積

バウンサー・ベビーチェア

便利でも床面積を使います。置きっぱなしにするか、折りたためるかを確認。

見る点
掃除の邪魔にならないか、使う時間帯

STORAGE RULE

収納を増やす前に見ること

01使う場所の近くに置く

おむつ替えセット、肌着、授乳用品は動線の近くに少量だけ。

02ストックを増やしすぎない

サイズアウトや肌に合わない可能性があるものは様子を見て買い足す。

03月齢で入れ替える

使わなくなった用品は早めに移動し、生活スペースを空ける。

SLEEP SPACE

ベビーベッドは「置けるか」と「安全確認」を分ける

狭い家では、ベビーベッドを置けるかだけでなく、柵の上げ下げ、収納扉のロック確認、周囲の通路幅まで含めて使えるかを見ます。消費者庁は、ベッドからの転落や収納扉に関する注意喚起を出しており、使用前には取扱説明書と公的情報を確認する前提で考えます。

設置予定場所で、柵・扉・引き出しを無理なく操作できるか

収納扉や柵のロックを毎回確認しやすい動線か

子供PSCマークなど、対象製品の安全制度を購入前に確認したか

レンタルや中古品は、基準・部品・破損・取扱説明書の有無を再確認できるか

安全性をこのサイトだけで判断せず、最新のメーカー説明、公的機関の注意喚起、リコール情報を確認してください。

PUBLIC INFO

買い物以外に確認すること

厚生労働省の「出産なび」では、母子健康手帳、妊婦健診、マタニティーマーク、妊娠中・産後に使える制度や相談支援などが案内されています。

助成内容や相談先は自治体により異なるため、住んでいる市区町村の案内も確認します。
