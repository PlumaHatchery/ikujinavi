# PROJECT.md - Parenting Goods Navi

## Goal

育児グッズを悩み別・カテゴリ別に整理し、読者が比較検討しやすい記事群を作る。

## Target Reader

最初の主対象は「都市部・車なし・収納少なめの家で、初めて出産準備をする家庭」。

- 初めて育児グッズを選ぶ保護者。
- 出産準備リストやランキングを見ても、何を基準に比較すればよいか分からない人。
- 車なし、狭い家、収納少なめ、ワンオペ時間が長いなど、生活条件に合う商品を探している人。
- 価格、使いやすさ、安全性、収納性だけでなく、置き場所・移動手段・買いすぎ防止も比較したい人。

詳しいペルソナは `PERSONAS.md` に置く。

## What This Workspace Does

- SEOキーワードを整理する。
- カテゴリと悩み別導線を作る。
- 商品比較メモを作る。
- 記事案と構成案を作る。
- 比較基準と注意表現を整える。

## What This Workspace Does Not Do Yet

- 外部サービス連携。
- GitHub push。
- Playwright本格実行。
- 大量スクレイピング。
- 価格・在庫の自動監視。

## Editorial Policy

医療・安全・育児アドバイスに関する断定は避ける。
商品価格、在庫、レビュー数、ランキングは変動するため、記事化前に必ず再確認する。
